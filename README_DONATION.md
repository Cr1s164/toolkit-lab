# 微信收款码配置说明

## 当前状态
- 网站已基本完成，等待收款码图片
- 占位符二维码已添加（SVG格式）
- 支持区域设计已完成

## 需要的图片
**微信收款码图片**（替换 `images/wechat-qr.svg`）
- 格式：PNG、JPG、SVG 均可
- 尺寸：建议 500×500 像素以上
- 要求：清晰可扫描，白底

## 提供方式
1. **直接发送图片文件**到 OpenClaw 会话
2. **提供图片URL**（网盘、图床等公开链接）
3. **告知本地路径**（在电脑上的具体位置）

## 自动检测
系统会自动检测 workspace 目录下的图片文件，如：
- `wechat-qr.png`
- `wechat-qr.jpg`  
- `qrcode.png`
- `收款码.png`

## 部署后的URL
网站部署后，收款页面将位于：`https://[domain]/#donate`

## 打赏统计（建议）
建议定期查看微信收款记录，跟踪：
- 单笔金额分布
- 每日/每周总额
- 高峰期时间
- 用户留言（如有）
